## Expense Tracker

Track your expenses using our script, it'll make sure you never spend an extra penny on anything ever again. Save, and control your spending; so you can buy that Lamborghini you always wanted... 100% real

## Usage

Download `expense_tracker.py` in `main` branch, then run `python3 expense_tracker.py` (based on python installation) in terminal.
